Introduction

The Archiver project consits of a main project, built in python with django, that holds smaller apps together. These apps, are easy to create and allow a robust set of features to be built around the data that the incase. These features include dashboard displays, a REST api framework, PostGIS database backend with the help of GEOdjango, and much more.

Getting Started

This project relies on a Python+Django backend to work. If you haven't used one (or either) before familiarize yourself with Python and Django.
